# 발생 배경 후보
- 자정 당시 지하주차장 차량 사고로 골반과 척추에 치명상을 입은 여성에게 공명이 발생했다.
- 파괴된 골격을 복구하는 과정에서 견갑골과 골반 주변의 조직이 추가 사지로 분화한다.

# 외형
- 본래의 두 팔과 두 다리 외에 견갑골과 골반에서 네 개의 추가 사지가 생성
- 총 여덟 개의 팔다리
- 벽과 천장에 몸을 고정할 수 있는 갈고리형 손발
- 거꾸로 매달린 상태에서도 정면을 보기 위해 크게 회전하는 목
- 긴 머리카락을 아래로 늘어뜨려 사람의 머리처럼 보이게 함

# 사냥 방식
- 천장과 배관에서 먹잇감을 관찰한다.
- 무리에서 떨어진 사람을 선택한다.
- 즉시 죽이기보다 팔다리를 손상시켜 이동 능력을 빼앗는다.
- 사체나 살아 있는 먹잇감을 천장의 영역으로 끌고 간다.
- 영역 경계에 사지를 배열해 다른 변이체에게 경고한다.

# 핵심 능력
- 여덟 개의 사지를 이용한 3차원 기동
- 천장, 벽, 배관을 통한 사각 공격
- 사수의 눈과 총구 방향 인식
- 여러 명의 팔다리를 동시에 절단
- 영역 안의 먹잇감을 즉시 죽이지 않고 이동 능력부터 제거

# 약점
- 불규칙한 다중 점멸광
- 반사면
- 견인줄과 셔터 모터
- 고압 증기
- 여러 사지를 동시에 구속하는 장치

# 이유진과의 관계
- 이유진의 지휘 신호를 거부한다.
- 자신의 영역에 들어온 이유진을 경쟁 지휘종으로 인식한다.
- 여왕거미와 이유진의 전투는 변이체가 하나의 군집이 아니라, 여러 지휘권과 영역으로 분열되어 있음을 보여준다.
