# 등급
일반 변이체의 인지 잔존형

# 특징
인간 시절 반복하던 행동이 운동 기억으로 남는다.
- 현관 비밀번호 입력
- 넥타이 정리
- 자동차 운전대 잡기
-  휴대전화 화면 확인
- 자녀의 이름 부르기

# 위험성
- 생존자는 이 행동을 보고 가족이 아직 인간이라고 착각한다.
- 하지만 행동은 의미 있는 의사소통이 아니라 남은 신경 회로의 반복일 가능성이 높다.

# 예외
- 이유진처럼 백신과 변이 인자가 동시에 작용한 지휘종은 실제 기억과 의도가 남을 수 있다. 초반에는 일반 잔존행동형과 구별하기 어렵다.
