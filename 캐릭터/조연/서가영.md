# 기본 정보
- 나이: 27세
- 계급: 육군 중사
- 직책: 군 산하 연구시설 의무·검체관리 부사관
- 첫 등장: 28화
- 1부 이후 역할: 성민 일행의 제한적 의료·군 정보 담당

# 성격
- 명령 체계에 익숙하지만 연구 대상자를 사람으로 보지 않는 시설 문화에 환멸을 느낀다.
- 이유진과 친한 사이는 아니었으나, 이유진이 피험자의 이름을 끝까지 기록했던 모습을 기억한다.
- 연구 전체를 알지는 못한다.
- 자신이 알고 있는 일부 정보가 오히려 더 위험한 오해를 만든다.

# 능력
- 검체 채취와 보관
- 감염 단계 판별
- 군 생물재난 코드와 장비 식별
- 방호복, 제독 장비 사용
- 지휘종 억제 장치의 기본 원리 파악

# 서사적 기능
- 이유진의 전문성을 대체하지 않는 하위 의료 인력.
- 군이 사태를 사전 인지했다는 증언자.
- 한기범과 성민 사이에서 군 내부의 분열을 보여준다.
- 31~50화 중 군에 재포획되거나, 성민 일행에게 남을지를 두고 갈등한다.
