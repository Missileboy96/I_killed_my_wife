# 기본 정보
나이: 27세
계급: 육군 중사
직책: 군 산하 연구시설 의무·검체관리 부사관
첫 등장: 28화

# 성격
명령 체계에 익숙하지만 피험자를 번호로만 취급하는 연구시설 문화에 환멸을 느낀다.
이유진과 가까운 사이는 아니었으나, 이유진이 피험자의 이름을 기록했던 모습을 기억한다.
연구의 전체 진실은 알지 못한다.
제한된 지식 때문에 잘못된 추론을 할 수 있다.

# 능력
검체 채취 및 보관
감염 단계 판별
군 생물재난 코드 식별
방호복과 제독 장비 운용
지휘종 억제기의 기본 원리 이해

# 제한
이유진의 전문성을 대체하지 않는다.
백신의 설계 원리나 진화의 돌의 본질을 완전히 알지 못한다.
성민의 피가 치료제인지 위험물인지 단정하지 못한다.
