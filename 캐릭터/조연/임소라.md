# 기본 정보
- 나이: 39세
- 직업: 회계사무소 직원
- 가족: 아들 유건
- 등장 및 사망: 9~10화

# 성격
- 아들을 보호하기 위해 감염 사실을 숨긴다.
- 타인의 아이가 감염됐다면 격리를 주장했겠지만, 자신의 아들은 예외라고 믿는다.
- 악인이 아니라 공포와 사랑 때문에 공동체를 위험에 빠뜨린다.

# 서사적 기능
- 가족애가 반드시 선한 결과를 만들지는 않는다는 점을 보여준다.
- 오수경과 대비된다.
    - 임소라: 아이를 위해 위험을 부정
    - 오수경: 아이를 위해 위험을 직시
