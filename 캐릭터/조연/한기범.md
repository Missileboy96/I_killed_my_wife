# 기본 정보
나이: 35세
계급: 육군 대위
소속: 생물학적 특수재난 대응부대
첫 등장: 30화
31~50화 역할: 추격자

# 성격
잔혹함을 즐기지는 않지만 명령을 거부하지 않는다.
자신이 국가를 지키고 있다고 믿는다.
민간인 사살 명령에 불편함을 느끼지만 임무를 우선한다.

# 특징
이유진을 이름이 아닌 E-01로 부른다.
변이 사태 이전부터 대응계획을 교육받았다.
실험의 전체 진실과 김수일의 정치적 계획은 모른다.
성민을 보호 대상이 아니라 생물학적 자산으로 본다.

# 장기 성장 가능성
31~50화 동안 성민 일행과 반복적으로 충돌
군 내부가 단일한 악의 조직이 아님을 보여줌
명령과 현장 사실의 괴리를 확인하며 점차 김수일 체제에 의문을 품음
