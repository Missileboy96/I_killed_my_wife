# 양아치
## 인적사항
- 이름 : 유민준
- 나이 : 23
- 직업 : 날건달
- 경력사항 : 깡패조직의 말단으로서 마약 운반책, 소위 던지기를 맡고 있었음.

## 특징
- 박정훈에게 연행당하고 있었음
- 깡패보다는 양아치에 가까운 존재임. 배짱이나 배포는 없고 간사한 존재임. 하지만 소시민적인 면모도 존재. 내면에는 선함이 존재함
- 고아출신으로서 보육원 출신들과 친함 -> 깡패가 되게 된 계기

## 아이디어
- 깡패가 된 계기후보1 : 와이프와 애가 있음. 돈나올 구멍이 없다.
- 깡패가 된 계기후보2-1 : 돈을 떼먹힌 업소녀의 빚을 대신 갚아주게 되어서, 하지만 업소녀는 지금 변이체가 됨. 
- 깡패가 된 계기후보2-2 : 공사당한 업소녀의 빚을 갚다가 깡패로서 일을 하게됨.
- 짬뽕 : 애가 생긴 것을 모르다가, 여자가 도망감.
- 알게된 계기 친구 or 실장 + 콜떼기?
- 여자친구를 최초에 팔아넘긴 깡패를 복수
