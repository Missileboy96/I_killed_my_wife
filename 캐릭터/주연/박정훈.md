# 형사
- 이름 : 박정훈
- 나이 : 50
- 직업 : 형사
- 경력사항 : 전형적으로 깡패와 유착한 부패 경찰임. 

# 특징
- 부패의 이유는 이미 죽었지만, 병에 걸린 딸을 위해 병원비를 마련하기 위해서였음. 딸이 죽었음에도 부패경찰로 남아 있는 이유는, 자포자기한 심정과 관성 때문.
- 이 때문에 차후 깡패 캐릭터와 엮일 가능성.
- 이야기가 진행되며 지금 시점에는 자신의 죽은 딸과 비슷한 나이일 김민서에게 애착을 가짐.
